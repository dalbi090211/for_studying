요구분석
: 오픈소스로 제공되는 blcokly라이브러리를 활용해 비전공자, 입문자에게 블록을 활용해 쉽게 코드를 작성하는 경험을 제공하고 
비전공자에게는 관련 지식이 없더라도 쉽게 프로그램을 작성할 수 있게 돕고 입문자에게는 복잡한 프로그램을 블록을 활용해
직관적이게 구조를 보여줌으로써 이해하기 쉽게 만든다.
 먼저 가장 실무적이지만 언어가 더러운 vba와 기초적인 c를 중심으로 페이지를 만들고 이후 예시창 구현이나 
코드 컴파일을 페이지에서 할 수 있게 하는 등 부가적인 기능을 구현한다.
최종적으로는 사용자가 많이 활용한 블록을 most카테고리에 띄운다거나 자주 작성되는 프로그램 예시를 제시하는등의 내가 직접 실습하면서 배우고 싶었던
서버에 데이터를 넘기는 기능과 ai관련 기능을 추가한다.

설계
: index, traslation, vba, c 4개의 페이지. 관습에 맞게 index에서 다른 페이지로 이동할 수 있게 처리하지만 
index에서는 펼쳐지는 사이드바, 이후 페이지에서는 펼쳐진 사이드바를 활용하여 자유롭게 이동할 수 있게 한다.
